RetryPolicy

open class RetryPolicy(retryPredicateDelegate: RetryPredicate, delayStrategyDelegate: DelayStrategy) : RetryPredicate, DelayStrategy

Represents a retry policy that combines a RetryPredicate and a DelayStrategy. The RetryPolicy class delegates its retry predicate and retry delay behaviors to the specified delegates. This allows for flexible and composable retry policies.

Since

1.0.0

Parameters

retryPredicateDelegate

The RetryPredicate delegate used to evaluate whether a retry should be performed.

delayStrategyDelegate

The DelayStrategy delegate used to determine the delay before a retry is attempted.
